# 产品介绍

#### 📖 简介

玩胶片和拍立得的乐趣，从囤积相纸开始。但面对越来越多、批次各异的存货，你是否经常忘记它们的买入价格，甚至错过有效期，失去了最佳显影期？

我的相纸库存 是一款专为摄影爱好者打造的轻量级库存管理工具。我们帮你把杂乱的囤货转化为清晰的数据，从入库、盘点、消耗到过期预警，提供一站式、数字化的相纸管理体验。让你的精力回归创作，库存琐事交给我们。

#### ✨ 核心功能亮点

* ⚡️ 极速入库，告别繁琐 内置丰富的相纸预设库，同时支持自定义添加。无论是拍立得宽幅还是 135/120 胶片，几步点击即可完成录入，大幅减少重复打字。
* 📦 批次管理，清晰溯源 相纸不仅是耗材，更是资产。按批次独立记录数量、买入价格、生产批号及过期时间，每一盒相纸的来源和价值都一目了然。
* 🔍 智能检索与多选批处理 库存太多找不到？通过多维度的筛选与排序（按类型、临期状态等），精准定位目标。支持多选批次进行集中操作与数据导出，盘点库存从未如此轻松。
* 📖 沉淀每一次“咔嚓”的使用记录 库存数量的减少不只是数字的扣除。你可以将变动与“使用”行为深度关联，记录下这次消耗是拍了哪场旅行或哪个重要的人，让冰冷的库存数据充满回忆。
* ⏰ 智能过期提醒，拒绝浪费 相纸价格高昂，错过保质期实在可惜。灵活设置提醒策略，提前感知即将过期的批次，帮你规划拍摄节奏，把每一张相纸都用在最佳状态。
* 🖼️ 一键生成库存海报 想在社交媒体上展示你的”弹药库”？支持一键导出精美的垂直排版库存海报。你可以自由配置导出内容，搭配标志性的 8x5 网格布局，让你的分享更具格调。
* 📸 生活拍立得相册 记录每一次拍摄的故事。选择使用的胶片和相机，关联拍摄日期、地点与标签笔记，还能附上照片。支持多种展示模式和海报导出，让你的拍摄回忆不再只是一串数字。
* 📷 相机管理 管理你的相机设备库。内置丰富的相机预设型号，支持自定义录入。记录每台相机的购买价格、购买日期和支持的胶片类型，全面掌握你的器材资产。
* 💰 卖出记录 除了”使用”，还支持记录”卖出”。关联售价、卖出渠道和目标，自动计算盈亏，让你清楚每一笔交易的收益。
* 📊 多维度统计分析 从库存总量、资产估值到各类型价值占比，再到过期明细与销售利润，一目了然地掌握你的相纸资产全貌。支持按官方价、市场价或进价进行估值，图表可视化展示。
* 🔄 数据导入导出 支持 JSON 格式的库存数据导入导出，方便备份恢复和跨设备迁移。还支持批量导入市场参考价格，实时掌握相纸行情。
